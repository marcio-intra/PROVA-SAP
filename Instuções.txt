Para rodar o script no terminal Bash, siga estes passos:

### 1. Salve o script em um arquivo
Primeiro, copie todo o conteúdo do script que forneci e salve em um arquivo chamado criar_projeto.sh. Você pode fazer isso de duas formas:

*Opção A - Usando um editor de texto:*
bash
nano criar_projeto.sh

Cole o conteúdo (Ctrl+Shift+V no terminal), salve (Ctrl+O) e saia (Ctrl+X)

*Opção B - Usando echo (recomendado):*
bash
echo '#!/bin/bash

# [Todo o conteúdo do script aqui...]' > criar_projeto.sh


### 2. Dê permissão de execução
bash
chmod +x criar_projeto.sh


### 3. Execute o script
bash
./criar_projeto.sh


### 4. (Opcional) Se você quiser ver o progresso em tempo real
bash
bash -x criar_projeto.sh


### Observações importantes:

1. Certifique-se de que você tem os requisitos instalados:
   - Node.js (v14+)
   - npm (v6+)

2. O script criará:
   - Uma pasta com o nome do projeto (prova-sap-marcio)
   - Todos os componentes e estrutura de arquivos especificados
   - Um arquivo documentacao.txt com as informações do projeto

3. Após a execução, entre na pasta do projeto e inicie o servidor de desenvolvimento:
bash
cd prova-sap-marcio
npm run dev


4. Se encontrar erros de permissão, você pode tentar executar com sudo (embora não seja o ideal):
bash
sudo ./criar_projeto.sh


5. Se você estiver no Windows usando WSL (Windows Subsystem for Linux), o processo é o mesmo.

O script foi projetado para ser auto-suficiente e criar todo o ambiente do projeto conforme suas especificações.
